15차시: 성적 계산 도우미

예제

def grade_of(score):
    if score >= 90: return "A"
    if score >= 80: return "B"
    if score >= 70: return "C"
    if score >= 60: return "D"
    return "F"
scores = [88, 92, 73, 64, 95]
avg = sum(scores) / len(scores)
grades = [grade_of(s) for s in scores]
print("점수:", scores)
print("평균:", round(avg, 1))
print("등급:", grades)

  2. 과목명과 점수를 딕셔너리로 출력
  3. 최고/최저 과목 함께 출력
