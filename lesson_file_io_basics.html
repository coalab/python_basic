12차시: 파일 입출력 (로컬 권장)

예제

fname = "hello.txt"
with open(fname, "w", encoding="utf-8") as f:
    f.write("안녕하세요! 파일 입출력 예제입니다.\n")
with open(fname, "r", encoding="utf-8") as f:
    content = f.read()
print("파일 내용:", content)
