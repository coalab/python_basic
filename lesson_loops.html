3차시: 반복문 (for / while)

핵심

  • for: 정해진 횟수만큼 반복
  • while: 조건이 참인 동안 반복

예제

# for: 1~5 출력
for i in range(1, 6):
print("for 반복:", i)


# while: 5~1 출력
n = 5
while n > 0:
print("while 반복:", n)
n -= 1

마무리 문제 (3문제)

  1. 1~10 합계를 for로 구하기
  2. while로 구구단 3단 출력
  3. 리스트 [3,6,9]를 for로 순회하며 합 출력
